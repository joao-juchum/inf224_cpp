\doxysubsection{Detailed Description}
Represents a group of multimedia objects. 

The \doxylink{classGroup}{Group} class inherits from {\ttfamily std\+::list} of shared pointers to {\ttfamily \doxylink{classMultimidia}{Multimidia}} objects. It allows grouping multiple multimedia objects and provides methods to interact with them.
