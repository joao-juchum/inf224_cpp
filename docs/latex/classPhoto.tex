\doxysubsection{Detailed Description}
Represents a photo with geolocation data. 

The \doxylink{classPhoto}{Photo} class extends the \doxylink{classMultimidia}{Multimidia} class by adding latitude and longitude information. It provides methods to access and modify these geolocation attributes, as well as methods to display and play the photo.
