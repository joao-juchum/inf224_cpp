\doxysubsection{Detailed Description}
Represents a video with a duration. 

The \doxylink{classVideo}{Video} class extends the \doxylink{classMultimidia}{Multimidia} class by adding a duration attribute. It provides methods to access and modify the duration, as well as methods to display and play the video.
