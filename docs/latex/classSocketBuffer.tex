\doxysubsection{Detailed Description}
Preserves record boundaries when exchanging messages between connected TCP/\+IP sockets. 

Ensures that one call to \doxylink{classSocketBuffer_afa3a2f239eb56c2e4fd4fa465f7fb54d}{read\+Line()} corresponds to one and exactly one call to \doxylink{classSocketBuffer_a96ba6ada0c8b57eacff2aa2e4e34c282}{write\+Line()} on the other side. By default, \doxylink{classSocketBuffer_a96ba6ada0c8b57eacff2aa2e4e34c282}{write\+Line()} adds ~\newline
 at the end of each message and \doxylink{classSocketBuffer_afa3a2f239eb56c2e4fd4fa465f7fb54d}{read\+Line()} searches for ~\newline
, \textbackslash{}r or ~\newline
\textbackslash{}r so that it can retreive the entire record. Beware messages should thus not contain these charecters.

Read a message from a connected socket. 

\doxylink{classSocketBuffer_afa3a2f239eb56c2e4fd4fa465f7fb54d}{read\+Line()} receives one (and only one) message sent by \doxylink{classSocketBuffer_a96ba6ada0c8b57eacff2aa2e4e34c282}{write\+Line()} on the other side, ie, a call to \doxylink{classSocketBuffer_a96ba6ada0c8b57eacff2aa2e4e34c282}{write\+Line()} corresponds to one and exactly one call to \doxylink{classSocketBuffer_afa3a2f239eb56c2e4fd4fa465f7fb54d}{read\+Line()} on the other side. The received data is stored in {\itshape message}. This method blocks until the message is fully received.

\begin{DoxyReturn}{Returns}
The number of bytes that were received or one of the following values\+:
\begin{DoxyItemize}
\item 0\+: shutdown\+Output() was called on the other side
\item \doxylink{classSocket_a9f68308228badcdd299cd83e62e36976a86c20a6b52053a1f39f5d31b28b9e7cb}{Socket\+::\+Failed} (-\/1)\+: a connection error occured
\item \doxylink{classSocket_a9f68308228badcdd299cd83e62e36976ae3fcc87ee9b909ea92c751f70df4c9ac}{Socket\+::\+Invalid\+Socket} (-\/2)\+: the socket is invalid. 
\end{DoxyItemize}
\end{DoxyReturn}
\begin{DoxyNote}{Note}
the separator (eg ~\newline
) is counted in the value returned by \doxylink{classSocketBuffer_afa3a2f239eb56c2e4fd4fa465f7fb54d}{read\+Line()}. 
\end{DoxyNote}
